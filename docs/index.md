# Parmesan Documentation

This site is the operator and engineering guide for Parmesan as it exists in
this repository.

## Quick Map

If you are new to the repo:

- start with [Getting Started](./getting-started.md)
- keep [Configuration](./configuration.md) open while editing config or agent files
- use [Policies](./policies.md) when changing behavior
- use [Operations / Dashboard](./operations-dashboard.md) when operating or debugging live sessions

The getting-started and operations guides include current dashboard screenshots
where a visual reference is more useful than another block of prose.
